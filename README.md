# 维艺轩家居 - 欧式高端家具艺术殿堂

## 项目简介

维艺轩家居是一个专注于欧式高端家具展示的现代化网站，采用优雅的设计风格和流畅的用户体验，展现雍容华贵的品牌形象。

## 网站特色

### 🎨 设计风格
- **欧式古典美学**：采用金色（#d4af37）作为主色调，体现高端奢华
- **优雅字体**：使用 Playfair Display 和 Cormorant Garamond 字体，营造古典艺术氛围
- **渐变背景**：精心设计的渐变色彩，增强视觉层次感
- **响应式设计**：完美适配各种设备尺寸
